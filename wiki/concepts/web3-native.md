## definition
Web3-native refers to applications, platforms, and technologies that are built on decentralized web principles. This includes the use of blockchain technology, decentralized protocols, and harnessing user-owned data, which empowers individuals rather than centralized entities.

## why-it-matters
In the context of OpenHome, web3-native concepts provide a framework for enhancing user autonomy and privacy. By leveraging decentralized technologies, OpenHome can promote a model of data ownership that aligns with contemporary digital rights movements. This ethos fosters a community where users can engage more meaningfully without the constraints imposed by traditional web architectures.

## how-it-manifests
Web3-native features might be integrated into OpenHome through elements such as decentralized identity systems, smart contracts for automating transactions, and community governance structures that allow users to participate actively in decision-making processes. These aspects not only enhance user experience but also cultivate trust and transparency within the platform.
