## definition
The Grant Program is an initiative by OpenHome to support developers contributing to the platform by providing resources, funding, and physical development kits.

## why-it-matters
This program fosters innovation and collaboration within the OpenHome community, encouraging developers to create new functionalities and improve the platform. By incentivizing contributions, the Grant Program helps sustain an ecosystem of creative applications and skills for OpenHome's voice AI agents.

## how-it-manifests
The Grant Program is visible in the allocation of physical development kits to top contributors, which can be used to create and prototype new abilities for the OpenHome platform. Announcements related to the program include opportunities for developers to apply for these kits and showcase their projects.
