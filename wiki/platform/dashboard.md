## what-it-is
The OpenHome Dashboard is an online interface that allows users to create and manage AI agents, known as Personalities, on the OpenHome platform. It serves as a central hub for interacting with various features, including the integration of Abilities that enhance the functionality of these agents.

## how-it-works
Users access the Dashboard through a web application at [app.openhome.com](https://app.openhome.com). Within this interface, individuals can set up their AI agents by defining attributes such as name and voice. Additionally, users can add or manage Abilities—modular voice plugins that enable the agents to perform tasks ranging from controlling smart devices to providing information. The Dashboard is designed to be intuitive, facilitating both novice and experienced developers to prototype and deploy their AI applications effectively.
