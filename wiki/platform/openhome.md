## what-it-is
OpenHome is an open-source Voice AI platform designed for building conversational AI agents known as Personalities. These agents can be enhanced through modular voice plugins called Abilities, which add new skills and functionalities.

## how-it-works
Users can create Personalities by configuring parameters such as name and voice, and then extend their capabilities by integrating various Abilities that allow interactions like calling APIs, playing music, and controlling smart devices. Abilities are activated through voice commands known as hotwords, enabling a dynamic and interactive user experience. The platform supports developers with comprehensive documentation and a dashboard for managing their agents, as well as access to community-built Abilities through GitHub.
