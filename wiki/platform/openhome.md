## what-it-is
OpenHome is an open-source Voice AI platform designed for building conversational AI agents known as Personalities, which can be enhanced with modular plugins called Abilities.

## how-it-works
OpenHome allows users to create AI agents, referred to as Personalities, by customizing aspects such as voice and persona. Abilities, which are plugins that add various skills to the agent, can be integrated to enable functionalities like API calls, music playback, device control, and more. These Abilities are activated through specific hotwords during voice interactions and can execute custom Python logic. The platform aims to facilitate intuitive voice-first applications that enhance the interaction between users and technology.
