## what-it-is
OpenHome is an open-source Voice AI platform that allows users to build conversational AI agents called Personalities. These agents can be enhanced with modular voice plugins known as Abilities, which add various skills and functionalities.

## how-it-works
The OpenHome platform enables developers to create and customize AI agents, referred to as Personalities, by setting up their name, voice, and conversational style. Users can enhance these agents with Abilities, which function as plugins that provide new skills such as calling APIs, playing music, and controlling smart home devices. Abilities are triggered during voice conversations through hotwords, allowing for interactive and dynamic user experiences. The platform provides a Dashboard for managing Personalities and exploring community-contributed Abilities, fostering collaboration and innovation.
