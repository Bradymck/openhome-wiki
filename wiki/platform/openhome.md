## what-it-is
OpenHome is an open-source Voice AI platform designed for creating conversational AI agents, known as Personalities, which can be enhanced using modular voice plugins termed Abilities.

## how-it-works
The OpenHome platform allows users to develop AI agents with distinct personalities by configuring their voice, persona, and conversation style. Users can add Abilities to these agents to equip them with various skills, such as calling APIs, playing music, or controlling smart home devices. Abilities are activated through hotwords during voice interactions and leverage custom Python logic for functionality. Developers can get started with comprehensive documentation available through the community resources, which include a dashboard for agent configuration and a GitHub repository for sharing and discovering Abilities.
