## what-it-is
The Live Editor is a component of the OpenHome platform that allows developers to create and edit abilities for conversational AI agents, facilitating real-time modifications and testing within the OpenHome ecosystem.

## how-it-works
The Live Editor provides an interactive interface where users can develop plugins, known as Abilities, that enhance the functionality of Personalities — the voice AI agents on the OpenHome platform. Developers can input and modify Python scripts that define the behavior of these Abilities, providing features such as API calls, music playback, or smart device control. Once changes are made in the editor, they can be tested immediately, enabling rapid iteration and enhancement of functionalities.
